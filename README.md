📦 Order Manager API

API REST para gerenciamento de pedidos, construída com Spring Boot, com foco em boas práticas de desenvolvimento backend e arquitetura escalável.

🚀 Objetivo

Este projeto tem como objetivo simular um sistema de gestão de pedidos, permitindo o controle de clientes, produtos e pedidos.

Também serve como base prática para evolução em tecnologias amplamente utilizadas no mercado Java, como Spring, JPA e testes automatizados.

🛠️ Tecnologias
Java
Spring Boot
Maven
Git
📂 Estrutura do Projeto

O projeto segue a organização padrão de aplicações Spring Boot, preparado para evolução com arquitetura em camadas:

src/main/java/
 ├── controller   → Camada de entrada (REST)
 ├── service      → Regras de negócio
 ├── repository   → Acesso a dados
 ├── model        → Entidades
